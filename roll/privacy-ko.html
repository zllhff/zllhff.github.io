개인정보 처리방침

최종 업데이트: 2026-05-22 · 즉시 발효.

Roll은 전적으로 사용자의 iPhone에서 작동합니다. 네트워크 호출을 일절 하지 않습니다. 저희는 사용자의 사진이나 사용자에 관한 어떠한 데이터도 수집, 전송, 저장하지 않으며, 접근 권한 자체가 없습니다.

수집하는 데이터

없습니다. Roll에는 서버, 분석 도구, 충돌 보고, 광고 ID, 사용 텔레메트리가 없습니다. Roll 계정도 존재하지 않습니다 — Roll에는 "사용자"라는 개념 자체가 없기 때문입니다.

제3자에게 전송하는 데이터

없습니다. 기기를 떠나는 유일한 신호는 Apple이 모든 iOS 앱에서 집계하는 익명 MetricKit 요약뿐이며, 저희는 이를 결코 받아보지 않습니다.

기기에 저장되는 항목

  • Apple Vision 프레임워크가 기기에서 직접 계산한 시각적 특징 인덱스(선명도, 얼굴, 임베딩). 사진 자체나 썸네일은 절대 저장되지 않습니다.
  • 작업 기록(어떤 세트를 검토하고, 보관하고, 버렸는지). 앱 내 "최근 버린 항목" 로그를 구동합니다.
  • 활동 로그 — 설정 → 개인정보 보호 및 데이터 → 활동 로그에서 확인할 수 있는 세션별 집계 기록. 총 개수만 포함하며 기기를 절대 떠나지 않습니다.
  • 사용자 설정(기본 Keep-N, 테마 등)은 iOS 표준 저장소에 보관됩니다.

위의 모든 항목은 설정 → Roll 재설정에서 언제든지 삭제할 수 있습니다.

사진 보관함 접근

Roll은 첫 실행 시 사용자가 부여한 권한 범위 내에서 Apple의 PhotoKit을 사용합니다. 제한된 접근이 권장 옵션입니다 — Roll은 사용자가 명시적으로 선택한 사진만 볼 수 있습니다. iPhone 설정 → Roll → 사진에서 언제든지 변경할 수 있습니다.

앱 내 구입

Roll은 두 가지 구입 옵션을 제공합니다: 일회성 Lifetime 구입과 주간 구독입니다. 둘 다 Apple의 StoreKit이 전적으로 처리하며, 저희는 결제 정보나 신원 정보를 전혀 받지 않습니다. 구독은 iPhone 설정 → Apple ID → 구독에서 취소하기 전까지 자동으로 갱신됩니다.

아동

Roll은 아동을 대상으로 하지 않으며, 아동의 데이터를 포함해 어떠한 데이터도 수집하지 않습니다.

직접 확인하는 방법

에어플레인 모드를 켜고 Roll을 평소처럼 사용해 보세요. 모든 기능이 그대로 작동합니다. Roll이 네트워크로 통신하지 않는다는 가장 간단하고 확실한 증거입니다.

지역별 고지

EU (GDPR)

Roll은 개인 데이터를 전혀 받지 않으므로 처리하지도 않습니다. GDPR이 정의하는 모든 권리(열람, 삭제, 이동)는 정의상 이미 보장됩니다 — 데이터는 사용자의 기기에 있으며 사용자가 직접 통제합니다.

캘리포니아 (CCPA / CPRA)

Roll은 개인정보를 판매하거나 공유하지 않습니다. 판매할 데이터 자체가 없으므로 별도의 거부(옵트아웃) 절차도 없습니다.

본 방침의 변경

Roll이 수집하거나 전송하는 내용이 변경되는 경우, 본 방침을 업데이트하고 변경이 적용되기 전에 앱 내에 알림을 표시합니다.
